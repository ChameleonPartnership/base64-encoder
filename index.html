About Base64

                

                    

                        
What is Base64?

                        
Base64 is a binary-to-text encoding scheme that converts binary data into a 64-character ASCII format. It's widely used in email, web APIs, and data transmission.

                    

                    

                        
Common Uses

                        
    
                            
  • Encoding binary files for email transmission
    • 
                            
  • Data-URIs in CSS and HTML
    • 
                            
  • API authentication tokens
    • 
                            
  • JWT and SAML tokens
    • 
                            
  • Secure data transmission
    • 
                        

                    

                    

                        
Encoding Size

                        
Base64 increases data size by ~33%. For example, 3 bytes of binary data becomes 4 Base64 characters.
